# Project Title
MRBUGGY7 AUTOMATED API TESTING

## Getting Started
This project requires Python3. and an additional requests module
```
pip install requests
```

Start by running runner.py

### Prerequisites
MRBUGGY7
```
http://mrbuggy.pl/mrbuggy7/
```
With first account manually made as:
```
Admin
```
